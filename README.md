# LinguaFlow

Қазақ пайдаланушыларға арналған ағылшын тілін үйрететін интерактивті платформа.

## Сілтемелер

- Сайт: https://interactive-english-learning-platfo.vercel.app
- Backend: https://interactive-english-learning-platform.onrender.com

## Технологиялар

- React, Vite, React Router, Axios
- Node.js, Express, JWT, Bcrypt, Multer
- PostgreSQL (Neon)
- Vercel + Render арқылы деплой жасалды

## Функциялар

- A1, A2, B1 деңгейлерінде 12 сабақ
- Қазақ мәдениеті тақырыбындағы Story сабақтар
- XP жүйесі, деңгейлер және жетістіктер
- Аватар жүктеумен жеке профиль
- Рейтинг кестесі
- Студент статистикасы бар Админ панелі
- Рөлдері бар JWT авторизация
- Мобильді адаптивті дизайн

## Локалды іске қосу
```bash
# Backend
cd server
npm install
npm run dev

# Frontend
cd client
npm install
npm run dev
```
